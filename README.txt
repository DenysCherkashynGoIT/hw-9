Создан интерфейс Findable и абстрактный клас Finder, определяющий базовую реализацию методов интерфейса.
Созданы классы-наследники от Finder: PhoneFinder, UserFinder, WordsCounter.

При инициализации экземпляра класса Finder или его подкласса  конструктор принимает обязательный параметр -
ссылку на файл-источник или объект типа File:
Finder(File source)
Finder(String source)

Классы наследники содержат  следующие приватные поля:
 private File source - хранит считываемый файл-источник.
 private <T> matches - хранит найденные значения-совпадений после последнего выполнения метода find(). Тип возвращаемых
                        данных <T>: ArrayList<String> для PhoneFinder, UserFinder и LinkedHashList для WordsCounter.
//--------------------------------------------------------------------------------------------------------------------------

Публичные методы класса:

       void setSource (File source)
- возвращает файл-источник как объект File

        File getSource ()
- возвращает файл-источник как объект File

        <T> getMatches ()
- возвращает найденные значения-совпадений после последнего выполнения метода find(). Тип возвращаемых данных <T>:
    ArrayList<String> для PhoneFinder, UserFinder и LinkedHashList для WordsCounter.

        Finder find ()
- віполняет сооветсвующий текущему классу поиск по файлу. Возвращает ссылку на текущий объект класса Finder

        void print ()
- выводит форматированный результат последнего поиска в консоль

        File printToFile(File destination)
        File printToFile(String destination)
- записывает форматированный результат последнего поиска в заданный файл по пути destination.
